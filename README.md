***# Spring Security JWT Login***



***## Description***

***This project implements JWT-based authentication using Spring Boot.***



***## API***

***POST /login***



***## Sample Request***

***{***

&#x20; ***"username": "admin",***

&#x20; ***"password": "admin123"***

***}***



***## Sample Response***

***JWT token string***



***## Credentials***

***admin / admin123***



***## UI (Optional)***

***Open in browser:***

***http://localhost:8082/login.html***



***Enter credentials to get JWT token.***
